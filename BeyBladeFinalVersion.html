BeyBlade

Support This Game

For all Beyblade fans!

This is an early prototype browser game that mimics the physics of a physical bey-battle.

How to connect : Host ID will appear on the top right hand corner of the host's screen once a game is hosted. Share this ID with other players who wish to join. To join a hosted game, simply copy and paste the Host ID into the host ID field after "Join Game" button is clicked.

Have fun and share your valuable feedback. Cheers !

Update (7.3.2022) : Play against CPU added
